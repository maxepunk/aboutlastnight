# How to Play Page — Copy Update Design

**Date:** 2026-04-12
**Scope:** Update how-to-play.html copy to reflect current game mechanics from the updated quick reference document, while preserving the page's atmospheric marketing voice.

## Source of Changes

Updated "How to Play" PDF quick reference (NeurAI Systems // Recovery Protocol // Case #ALN-2026), used as the in-venue player handout. The webpage should reflect the same mechanical reality but in a marketing-appropriate register.

## Page Structure (Post-Update)

### 1. Hero — No Changes

Title, subtitle, background, and nav link unchanged.

### 2. Meet Your Character — Minor Copy Update

**Current:** "You'll receive a character sheet — your name, some memory fragments you still recall, maybe the name of someone you remember knowing."

**Updated:** Reframe character sheet contents to "identity, relationships, and personal stakes" to match current game design. Keep the atmospheric tone — "You don't know everything about who you are yet. Neither does anyone else." and "This is your starting point" remain.

### 3. The Game — Mechanics Rework

**Three-step loop** (Unlock → Scan → Decide) stays, with copy tightened to match PDF:

- **Unlock:** "Work through Marcus's security measures to access memory tokens." (replaces "puzzle out how to get past the locks")
- **Scan:** "Read token contents using one of Marcus's memory display scanner prototypes." (minor rewording)
- **Decide:** "After scanning a token to read its contents, choose what happens to each memory you recover." (replaces "Memory tokens are your currency" framing)

**Action cards** — three cards instead of four, reordered to TRADE → EXPOSE → BURY:

- **TRADE:** "Negotiate directly with other players. Give, swap, leverage, or return a memory to its rightful owner." (expanded from old version, absorbs some NEGOTIATE concepts)
- **EXPOSE:** "Turn the memory in as evidence. Its summary is posted to the Evidence Board for all to see. Exposed memories can be cited in the group's final statement." (reframed around Evidence Board; drops anonymous/credit distinction)
- **BURY:** Expanded with "sell" framing: "Sell the memory to be permanently erased. Its contents are gone forever — no one can reference them again." Shell account vs personal account as distinct items. New closing line: "The more sensitive the information, the more valuable it is to NeurAI's board of directors to have it disappeared." (broader than just Marcus-incriminating tokens)

**Connective copy preserved after cards:**
- "That's the loop. But you're not alone in it."
- "Others are unlocking memories too..."
- "What is found, what is shared, what is hidden... all of that is the game."
- "The clock is always running."

### 4. The Group Statement — New Section

New section between The Game and The Morning After. Atmospheric wrapper around the endgame mechanic:

- Recovery window closes, transactions end, police are on their way
- Introduce the two public displays visible throughout the night: **Evidence Board** (summaries of exposed memories) and **Account Ledger** (account names and balances)
- Only memories on the Evidence Board can be cited as evidence; buried memories are gone
- The group must agree on one collective witness statement
- Characters have personal reasons for wanting certain outcomes — but the statement is collective
- Closing: "Find a story you can all live with before time runs out."

Styling: Same `explanation-section fade-in-section` pattern as other sections, with a background image (reuse one of the existing noir images).

### 5. The Morning After — Copy Update

**Keep:** Section name, "personalized" language, flavor text about catching missed narrative pieces.

**Update:** Name the Nova News blog specifically. Mention the link is emailed after the game. Tighten per PDF's cleaner framing of the post-game report.

### 6. Footer — No Changes

- **"Want Justice? Profit? Secrecy?"** line — redundant with dedicated Group Statement section
- **Anonymous/credit distinction** on EXPOSE — no longer part of the mechanic

## What's Preserved
